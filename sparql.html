Query SPARQL


PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?instrument ?label
WHERE {
  ?instrument a arco:MusicalInstrument ;
              rdfs:label ?label .
  FILTER(REGEX(?label, "liuto", "i"))
}
LIMIT 50

Questa query recupera gli strumenti musicali etichettati come "liuto" dal sito di ArCo (Archivio del Catalogo). Utilizza direttamente i dati pubblicati dal sito.
